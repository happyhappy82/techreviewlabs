<summary>Q1. 신소재공학과에서 VASP는 노트북으로 돌릴 수 있나요?</summary>

간단한 테스트 계산(원자 수 10~20개 수준)은 노트북에서도 가능해요. 하지만 본격적인 DFT 계산은 연구실 서버나 HPC 클러스터에서 돌리는 게 일반적이에요. 노트북은 입력 파일 작성, 결과 분석, 시각화 용도로 활용하는 경우가 많아요.


</details>

<details>
<summary>Q2. 신소재공학과 노트북에 GPU가 왜 필요한가요?</summary>

LAMMPS, GROMACS 같은 분자동역학 시뮬레이션은 CUDA 기반 GPU 가속을 지원해요. GPU가 있으면 CPU 대비 5~10배 빠르게 시뮬레이션을 수행할 수 있어요. 또한 Materials Studio의 3D 구조 시각화에도 GPU가 도움이 돼요.


</details>

<details>
<summary>Q3. RAM 16GB로 충분한가요, 32GB가 필요한가요?</summary>

학부 과정에서는 16GB로 대부분의 작업이 가능해요. MATLAB 행렬 연산, Python 데이터 분석, 기본적인 시뮬레이션 테스트에는 충분해요. 다만 대학원에서 대규모 원자 모델 시뮬레이션을 할 계획이라면 32GB 이상을 고려하는 게 좋아요.


</details>

<details>
<summary>Q4. 신소재공학과도 맥북을 쓸 수 있나요?</summary>

권장하지 않아요. VASP, Materials Studio 등 주요 재료 시뮬레이션 소프트웨어가 Windows 또는 Linux 기반이에요. MATLAB과 Python은 macOS에서도 잘 동작하지만, 전공 소프트웨어 호환성을 고려하면 Windows 노트북이 훨씬 안전한 선택이에요.


</details>

<details>
<summary>Q5. 게이밍 노트북을 신소재공학용으로 써도 되나요?</summary>

네, 오히려 좋은 선택일 수 있어요. 게이밍 노트북은 고성능 CPU와 NVIDIA RTX GPU를 탑재하고 있어서 시뮬레이션과 연산 작업에 적합해요. 다만 무게가 무겁고 배터리 수명이 짧은 점은 감안해야 해요.


</details>

<details>
<summary>Q6. XRD, SEM 분석 소프트웨어는 어떤 스펙이 필요한가요?</summary>
