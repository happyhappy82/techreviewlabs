<details>
<summary>CAN-Pro는 어떤 프로그램인가요?</summary>

한국영양학회에서 개발한 컴퓨터 영양평가 프로그램이에요. 식단을 입력하면 영양소 섭취량을 자동으로 계산하고, 한국인 영양섭취기준(DRI)과 비교해서 평가해줘요. 식품영양학과 학생이라면 거의 필수로 사용하는 프로그램이에요. Windows 전용이니 맥북 사용 시 가상머신이 필요해요.


</details>

<details>
<summary>식품영양학과에서 맥북을 써도 되나요?</summary>

CAN-Pro가 Windows 전용이라 맥북에서는 Parallels 같은 가상머신이 필요해요. SPSS는 macOS 버전이 있지만 기능 차이가 있을 수 있어요. 식품영양학과라면 Windows 노트북이 더 안전한 선택이에요.


</details>

<details>
<summary>RAM 8GB로도 충분한가요?</summary>

CAN-Pro와 Excel 정도는 8GB에서 가능하지만, SPSS에서 대용량 데이터셋을 분석하거나 여러 프로그램을 동시에 사용하면 부족해질 수 있어요. 16GB를 추천해요.


</details>

<details>
<summary>식품영양학과에서 코딩을 하나요?</summary>

최근에는 R이나 Python을 이용한 영양 데이터 분석 수업이 늘고 있어요. 하지만 전공 필수는 아닌 경우가 많고, SPSS와 Excel이 여전히 주력이에요. 대학원 진학을 고려한다면 R/Python을 미리 익혀두면 유리해요.


</details>

<details>
<summary>노트북 화면 크기는 어느 정도가 좋나요?</summary>

Excel 영양소 계산표를 볼 때는 15인치 이상이 편해요. 14인치도 사용 가능하지만 열이 많은 스프레드시트를 볼 때 좁게 느껴질 수 있어요. 16인치가 이상적이고, 휴대성을 중시하면 15.6인치가 적절해요.


</details>
