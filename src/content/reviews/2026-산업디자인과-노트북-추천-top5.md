산업디자인과 학생에게 노트북은 단순한 학습 도구가 아니라 디자인 작업의 핵심 장비예요. 제품 컨셉을 3D로 모델링하고, 실사 같은 렌더링 이미지를 만들고, 디자인 프레젠테이션 보드를 제작하는 모든 과정이 노트북에서 이루어지죠.


산업디자인과는 건축학과나 시각디자인과와 사용하는 소프트웨어가 다르기 때문에, 학과 특성에 맞는 노트북을 선택하는 것이 중요해요. Rhino와 SolidWorks가 중심이면서도 KeyShot이나 Blender로 렌더링까지 해야 하니, GPU와 CPU 모두 강력해야 해요.


## 산업디자인과 노트북 선택 기준


산업디자인과에서 주로 사용하는 프로그램과 작업이에요.

- **Rhino 3D (Rhinoceros)**: NURBS 기반 3D 모델링 (산디 필수 소프트웨어)
- **SolidWorks**: 기구적 3D CAD 설계 (Windows 전용)
- **KeyShot**: 실시간 렌더링 (GPU 또는 CPU 렌더링)
- **Blender**: 무료 3D 모델링/렌더링/애니메이션
- **Adobe Illustrator / Photoshop**: 2D 디자인 및 프레젠테이션 보드
- **Grasshopper (Rhino 플러그인)**: 파라메트릭 디자인

필요한 스펙은 다음과 같아요.

- **CPU**: Intel Core i7/Core Ultra 7 이상 또는 AMD Ryzen 7 이상
- **RAM**: 최소 16GB, 권장 32GB
- **GPU**: NVIDIA RTX 4050 이상 (외장 그래픽 필수)
- **SSD**: 512GB 이상 (모델링 파일 + 렌더링 결과물 저장)
- **디스플레이**: 15인치 이상, 색재현율 DCI-P3 90% 이상
> SolidWorks는 Windows 전용이에요. 학교에서 SolidWorks를 필수로 사용한다면 반드시 Windows 노트북을 선택해야 해요.

## 구매 가이드: 산업디자인과 핵심 체크포인트

1. **GPU + 색재현율이 가장 중요해요**: KeyShot, Blender 렌더링에는 외장 GPU가 필수이고, CMF 작업에는 정확한 색재현율(최소 sRGB 100%, 권장 DCI-P3 90%+)이 필요해요.
2. **RAM 32GB 권장**: Rhino + Grasshopper + KeyShot을 동시에 열면 16GB로는 부족할 수 있어요.
3. **SolidWorks 사용 시 Windows 필수**: SolidWorks는 Windows 전용이에요. 학교에서 필수로 사용한다면 맥북은 선택지에서 제외하세요.
4. **화면 크기 15인치 이상**: 3D 모델링 작업은 화면이 클수록 유리해요. 최소 15인치, 가능하면 16인치를 선택하세요.
5. **터치스크린/스타일러스 펜 지원**: Rhino에서 스케치하듯이 제품 형태를 그릴 때 터치스크린이 유용해요.
