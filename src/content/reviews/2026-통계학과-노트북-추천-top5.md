<details>
<summary>통계학과에서 R과 Python 중 뭐가 더 많이 쓰이나요?</summary>

전통적인 통계 수업에서는 R을 더 많이 써요. 하지만 최근에는 데이터 사이언스 커리큘럼이 확대되면서 Python 비중이 높아지고 있어요. 두 언어 모두 배우게 되니, 둘 다 원활하게 돌릴 수 있는 노트북이면 좋아요.


</details>

<details>
<summary>SAS를 꼭 써야 하나요? R이나 Python으로 대체할 수 있나요?</summary>

SAS는 금융권, 제약, 대기업에서 많이 쓰이기 때문에 학과에서 가르치는 경우가 많아요. 학술 연구에서는 R이 더 많이 쓰이고, 산업 현장에서는 SAS와 Python이 혼용돼요. SAS University Edition은 무료로 사용할 수 있으니 노트북 사양 부담은 크지 않아요.


</details>

<details>
<summary>RAM 16GB면 데이터 분석에 충분한가요?</summary>

학부 수준의 데이터셋(수만~수십만 행)이라면 16GB면 충분해요. 하지만 수백만 행 이상의 대용량 데이터를 다루거나, R에서 여러 데이터프레임을 동시에 메모리에 올리면 부족할 수 있어요. 대학원 진학을 고려한다면 32GB를 추천해요.


</details>

<details>
<summary>통계학과에서 GPU가 필요한가요?</summary>

학부 과정에서는 거의 필요 없어요. R 통계 분석, SPSS, SAS 모두 CPU 기반이에요. 하지만 데이터 사이언스 복수전공이나 대학원에서 딥러닝을 한다면 NVIDIA GPU(CUDA 지원)가 도움이 돼요. 기본은 Google Colab으로 커버하고, 본격적으로 필요해질 때 GPU 노트북을 고려해도 돼요.


</details>

<details>
<summary>맥북으로 통계학과 수업을 들을 수 있나요?</summary>

R, Python, Jupyter Notebook은 macOS에서 잘 돌아가요. 하지만 SAS는 macOS 네이티브 지원이 제한적이고, SPSS도 일부 기능 차이가 있어요. Windows가 더 안전한 선택이지만, SAS를 가상머신이나 클라우드로 사용할 수 있다면 맥북도 괜찮아요.


</details>
